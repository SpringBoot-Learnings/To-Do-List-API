# 📝 To-Do List API

A simple and beginner-friendly Spring Boot API for managing your daily to-dos.

## 🔧 Features

- Create, Read, Update, Delete To-Do items
- Mark tasks as completed
- Partial updates using PATCH requests
- H2/MySQL database support
- JPA + Spring Data

## 📦 Tech Stack

- Java 17
- Spring Boot
- Spring Data JPA
- MySQL / H2
- Lombok

## 📁 Project Structure

com.sharanya.todo_api ├── controller ├── model ├── repository └── service

## 🚀 How to Run

1. Clone the repo:
   ```bash
   git clone https://github.com/SpringBoot-Learnings/To-Do-List-API.git
2. Navigate to project folder:
   ```bash
   cd To-Do-List-API
3. Build and run:
   ```bash
    mvn clean install
    mvn spring-boot:run
4. Test endpoints on http://localhost:8080/api/todos
